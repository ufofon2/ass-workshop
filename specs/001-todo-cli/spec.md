## 개요

작고 사용하기 쉬운 CLI 기반 TODO 관리 도구입니다. 사용자는 로컬에서 작업을 추가, 수정, 완료, 삭제하고 목록을 볼 수 있습니다. 목표는 단순성에 중점을 둔 생산성 도구를 제공하는 것입니다.

## 목표

- 빠르게 할 일 추가/확인/완료 처리
- 파일 기반의 쉬운 설치와 사용
- 개발자가 확장하기 쉬운 코드베이스

## 주요 기능

- `add`: 할 일 항목 추가
- `list`: 할 일 나열
- `done`: 항목 완료 표시
- `remove`: 항목 삭제

## 데이터 모델

- `Task`:
  - `id`: 고유 식별자
  - `title`: 문자열
  - `completed`: 불리언
  - `created_at`: 타임스탬프
  - `due` : 날짜 (선택)
  - `priority` : 낮음/중간/높음 (선택)

## 기술 컨텍스트

- 언어/버전: Python 3.11
- CLI 프레임워크: Typer
- 패키지/배포: pip / PyPI(선택)
- 스토리지: 로컬 JSON 파일 (예: `.todo.json`) — 단순하고 의존성이 없음
- 기본 출력 포맷: JSON (머신·스크립트 친화적)

## 입력/출력 형식

- TODO 항목 저장 포맷은 위의 스토리지 결정에 따라 달라집니다.
- `list` 명령의 기본 출력은 위의 기본 출력 포맷 결정에 따릅니다.

## 테스트

- 단위 테스트: pytest
- 통합 테스트: 간단한 CLI 시나리오 테스트 (자동화)
